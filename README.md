# Apply Direct

Apply Direct is a Vite + React + TypeScript web app for sending your EmailJS template email directly to a hiring email address.

## Features

- Single email field workflow
- Subject and body managed directly in your EmailJS template
- Direct EmailJS send flow
- Success and failure toasts
- Mobile responsive modern UI

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in your EmailJS keys.
3. In your EmailJS template, configure the recipient field using:
   - `to_email`

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
